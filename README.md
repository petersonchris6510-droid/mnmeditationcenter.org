# Minnesota Meditation Center

Official website for **Minnesota Meditation Center**, a Dhammakaya Buddhist temple located in Osseo, Minnesota. The site serves as the public-facing web presence for the center, providing information about meditation classes, schedules, location, and how to get involved.

**Live site:** [mnmeditationcenter.org](https://mnmeditationcenter.org)

---

## Pages

- **Home** (`index.html`) — Hero, welcome message, weekly schedule overview, first visit info
- **About** (`about.html`) — Dhammakaya tradition, mission, community overview
- **Schedule** (`schedule.html`) — Full class schedule with FAQ
- **Visit** (`visit.html`) — Address, directions, Google Maps, what to expect
- **Get Involved** (`get-involved.html`) — Volunteering, donations, ways to support

---

## Tech

- Pure HTML/CSS — no frameworks, no dependencies
- Hosted via GitHub Pages
- Custom domain managed through GoDaddy DNS
- Fonts: Montserrat + Nunito via Google Fonts
