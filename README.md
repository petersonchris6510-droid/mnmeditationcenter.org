# Minnesota Meditation Center — Website

Public website for [Minnesota Meditation Center](https://mnmeditationcenter.org) in Osseo, MN. Static HTML/CSS site hosted via GitHub Pages.

---

## Pages

| File | URL | Purpose |
|---|---|---|
| `index.html` | / | Homepage |
| `about.html` | /about | About the center and tradition |
| `schedule.html` | /schedule | Weekly class schedule |
| `visit.html` | /visit | Directions and visitor info |
| `team.html` | /team | Resident teachers |
| `temple.html` | /temple | Thai community and temple services |
| `get-involved.html` | /get-involved | How to connect and support |
| `th/index.html` | /th/ | Thai-language homepage (in progress) |

Geo-targeted landing pages are in the `/geo/` directory.

---

## Images

All images are in the `/images/` directory. To update an image, upload a new file with the same filename via the GitHub web interface and commit. The page will reflect the change immediately.

---

## How to Edit a Page

1. Open the file in GitHub (e.g., `about.html`)
2. Click the pencil icon (Edit this file)
3. Make your changes
4. Scroll down, add a brief commit message describing what you changed
5. Click **Commit changes**

The live site updates within a minute or two.

---

## How to Upload a New File

1. Navigate to the folder where the file belongs
2. Click **Add file > Upload files**
3. Drag in your file or browse to select it
4. Add a commit message and click **Commit changes**

---

## Key Notes

- This is a static site. There is no build step, no npm, no local setup required.
- Do not restructure the CSS or refactor page layouts without a specific reason. Content edits should stay isolated from structural changes.
- All meditation classes are listed as free. Do not add pricing language anywhere on the site.
- "Monk Choo" is the correct form of address in all on-site copy. Do not use "Choo" alone.
- The Thai pages (`th/`) use `hreflang` tags for SEO. Do not remove these.
